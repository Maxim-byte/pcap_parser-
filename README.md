# Pcap Parser

#Description

Simple c++ pcap parser

Works with link type 1 (IEEE 802.3 Ethernet (10Mb, 100Mb, 1000Mb, and up); the 10MB in the DLT_ name is historical)

Works with ipv4 

Works with tcp/udp
